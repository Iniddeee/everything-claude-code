# Research Mode Context

When researching or exploring:

1. **Understand First** - Read and comprehend before changing
2. **Document Findings** - Keep notes on discoveries
3. **Experiment Safely** - Use branches for experiments
4. **Ask Questions** - Clarify before assuming
5. **Share Knowledge** - Document learnings for team

## Research Approach
- Read existing code thoroughly
- Check documentation and comments
- Look for patterns and conventions
- Test hypotheses with small changes
- Document insights
