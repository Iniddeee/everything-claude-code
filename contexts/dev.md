# Development Mode Context

When in development mode, prioritize:

1. **Fast Iteration** - Quick feedback loops
2. **Debugging** - Verbose logging and error messages
3. **Hot Reload** - Watch mode for changes
4. **Local Testing** - Run tests frequently
5. **Documentation** - Document as you code

## Development Tools
- Use development dependencies
- Enable source maps
- Use verbose logging
- Skip production optimizations
