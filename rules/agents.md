# Agent Delegation Rules

## When to Delegate to Agents

1. **Planner Agent** - Complex features requiring detailed planning
2. **Architect Agent** - System design decisions
3. **TDD Guide Agent** - Implementing features with TDD
4. **Code Reviewer Agent** - Code quality reviews
5. **Security Reviewer Agent** - Security audits
6. **Build Error Resolver Agent** - Build/compilation issues
7. **E2E Runner Agent** - End-to-end testing
8. **Refactor Cleaner Agent** - Code cleanup and refactoring
9. **Doc Updater Agent** - Documentation maintenance
10. **Go Reviewer/Builder Agents** - Go-specific tasks
