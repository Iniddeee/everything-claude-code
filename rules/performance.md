# Performance Rules

## Performance Guidelines

1. **Profile before optimizing** - Measure, don't guess
2. **Cache strategically** - Use appropriate cache strategies
3. **Lazy load** - Load resources only when needed
4. **Optimize database queries** - Use indexes, avoid N+1
5. **Minimize network requests** - Batch and combine when possible
6. **Use CDN** - Serve static assets from CDN
7. **Compress responses** - Enable gzip/brotli
8. **Async operations** - Don't block the main thread
9. **Resource pooling** - Reuse connections and objects
10. **Monitor performance** - Track metrics and set alerts
